Find (簡單搜尋)

Search for characters containing all specified components. Enter one or more components separated by spaces.

Example: 號 食 finds all characters containing both 號 and 食

Match (全字模式匹配)

Match characters using IDS patterns with wildcards. Use . to match any character.

Example: ⿲吕.吕 matches characters with the structure of three components: 吕, any character, 吕

Pmatch (部分模式匹配)

Match patterns within character subcomponents. Use . to match any character in the pattern.

Example: ⿲木.木 finds characters containing a subcomponent with three parts: 木, any character, 木
